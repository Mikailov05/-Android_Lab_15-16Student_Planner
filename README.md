# Student Planner

## 1. Название проекта
**Student Planner** — Студенческий планер

## 2. Краткое описание приложения
Student Planner — это мобильное приложение для студентов, которое помогает организовать учебный процесс. Приложение позволяет просматривать список изучаемых дисциплин, получать детальную информацию о каждой дисциплине, а также управлять профилем и настройками приложения.

## 3. Список реализованных экранов
- **Главный экран (HomeScreen)** — отображает список всех учебных дисциплин в виде карточек с названием, преподавателем и текущей оценкой
- **Экран деталей (DetailsScreen)** — показывает подробную информацию о выбранной дисциплине: полное описание, преподаватель, количество баллов и текущая оценка
- **Экран профиля (ProfileScreen)** — содержит информацию о студенте: ФИО, группа, факультет, email, а также статистику успеваемости
- **Экран настроек (SettingsScreen)** — позволяет настраивать параметры приложения: push-уведомления, темную тему и другие опции

## 4. Используемые технологии
- **Kotlin** — язык программирования
- **Jetpack Compose** — современный инструментарий для построения пользовательского интерфейса
- **Navigation Compose** — библиотека для реализации навигации между экранами
- **Material Design 3** — дизайн-система для создания современного UI


## 5. Схема навигации между экранами
**Маршруты навигации:**

- `home` → переход на главный экран
- `profile` → переход в профиль студента
- `settings` → переход в настройки
- `details/{subjectId}` → переход на экран деталей выбранной дисциплины

**Навигация с параметрами:**

- При клике на карточку дисциплины вызывается `navController.navigate("details/$subjectId")`
- На экране деталей параметр извлекается через `backStackEntry.arguments?.getString("subjectId")`

**Управление Back Stack:**

- Кнопка "Назад" на всех экранах вызывает `navController.popBackStack()`
- Системная кнопка "Назад" возвращает на предыдущий экран в стеке
- На главном экране кнопка "Назад" закрывает приложение
